# Picture Puzzle Maker

A lightweight browser-based tool that turns any uploaded image into an interactive drag-and-drop puzzle.

Built using pure HTML, CSS, and JavaScript. No frameworks. No backend.

## Overview

Picture Puzzle Maker allows users to upload an image, select puzzle shape and difficulty, and generate a playable puzzle directly in the browser.

Designed for simplicity, smooth interaction, and dynamic puzzle generation.

---

## Features

- Upload any image
- Multiple puzzle shapes:
  - Square
  - Circle
  - Horizontal rectangle
  - Vertical rectangle
- Adjustable difficulty levels
- Drag-and-drop puzzle pieces
- Smart snapping system
- Group dragging when pieces connect
- Automatic completion detection

---

## Project Structure

- index.html — Main interface
- style.css — Layout and styling
- script.js — Puzzle logic and interactions

---

## Getting Started

1. Download or clone the repository
2. Open `index.html` in your browser
3. Upload an image
4. Create and solve your puzzle

No installation required.
